# Hilfe wenn du feststeckst

Analysiere das aktuelle Problem und finde eine Lösung.

## Anweisungen

1. Frage: "Was funktioniert nicht? Beschreib es in deinen Worten."
2. Klassifiziere das Problem:
   - **Knowledge Gap**: Wissen fehlt → erkläre und zeige Lösung
   - **Circular Debugging**: Im Kreis drehen → frischer Ansatz
   - **Scope Creep**: Vom Thema abgekommen → zurück zum Kern
   - **Environmental**: Server/Tool/Config Problem → systematisch prüfen
   - **Wrong Approach**: Falscher Lösungsansatz → Alternative vorschlagen
3. Suche in der SQLite DB ob dieses Problem schon mal gelöst wurde
4. Nutze den Unsticker-Agent für Root-Cause-Analyse wenn nötig
5. Präsentiere Optionen sortiert nach: Schnellste Lösung zuerst

## Format
Erkläre das Problem und die Lösung in einfacher Sprache.
Keine technischen Details die der User nicht braucht.
