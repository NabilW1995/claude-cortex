# Tagesbeginn-Ritual

Starte eine neue Arbeitssession.

## Anweisungen

1. Lies .claude/memory.md — was war der letzte Stand?
2. Lade die letzten 10 Learnings aus der SQLite DB
3. Lies die letzte Daily Note aus daily-notes/
4. Prüfe .env gegen .env.example — fehlen Variablen?
5. Zeige eine kurze Zusammenfassung:
   - Was wurde zuletzt gemacht?
   - Welche Tasks sind offen?
   - Welche Learnings sind relevant?
6. Frage: "Woran möchtest du heute arbeiten?"

## Format
Halte die Zusammenfassung kurz (max 10 Zeilen).
Nutze einfache Sprache — kein Fachjargon.
