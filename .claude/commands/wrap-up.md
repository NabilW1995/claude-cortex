# Tagesende-Ritual

Beende die Arbeitssession und bereite die nächste vor.

## Anweisungen

1. Erstelle eine Daily Note in daily-notes/YYYY-MM-DD.md:
   - Was wurde heute gemacht?
   - Was ist noch offen?
   - Welche Entscheidungen wurden getroffen?
   - Gibt es Blocker?
2. Aktualisiere .claude/memory.md
3. Prüfe Knowledge-Nominations — gibt es Learnings zum Promoten?
4. Erstelle einen Checkpoint-Commit wenn nötig
5. Zeige eine kurze Zusammenfassung für morgen

## Format Daily Note

```
# [Datum]
## Erledigt
- ...
## Offen
- ...
## Entscheidungen
- ...
## Nächste Schritte
- ...
```
